# Service as GPU

This project is a simple simulation of a Service-as-GPU system using Python and Flask.

## Description
Users can submit GPU jobs.
Each job has a status.
Jobs start as PENDING and after execution become COMPLETED.

This project focuses on understanding the core logic of a GPU job service.

## Features
- Create jobs
- List jobs
- Execute jobs (simulated)
- REST API using Flask

## Technologies
- Python
- Flask

## How to run
1. Install dependencies
2. Run app.py
3. Use API endpoints
